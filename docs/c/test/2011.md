9. 以下程序的输出结果是（C）<br/>
```c
int a = 1, b = 3, c = 0, d = 7, e = 9, f = 2, g = 4, h = 6, i = 8, j = 0,
        k = 10;
    a+=++b*c+--e/f-i%d?c--&++i?++b||--d:!j==1:k+++a;
    printf("a=%d", a);
```
A. a=0　　　　B. a=1　　　　C. a=2　　　　D. a=3<br/>

解：
```c
 int a = 1, b = 3, c = 0, d = 7, e = 9, f = 2, g = 4, h = 6, i = 8, j = 0,
        k = 10;
    a += ++b * c + --e / f - i % d ? c-- & ++i ? ++b || --d : !j == 1 : k++ + a;
```
4*0 + 8/2 - 8%7 不等于 0，0&9 等于 0，!j == 1 等于1，1+1 = 2
