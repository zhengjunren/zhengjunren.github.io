# What is a bug

bug是一个英文单词，本意是指昆虫、小虫、损坏、犯贫、缺陷、窃听器等意思。现在一般是指在电脑系统或程序中，隐藏着的一些未被发现的缺陷或问题，简称程序漏洞。另外bug还有一种引申意义，用来形容某事物厉害的超乎想象。
