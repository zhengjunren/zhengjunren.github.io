# Linux 安装 Tomcat

# 概述

此处以 Tomcat 8.5.23 为例

下载地址： https://tomcat.apache.org/

# 解压缩并移动到指定目录

# 解压缩

tar -zxvf apache-tomcat-8.5.23.tar.gz
1

# 进入文件夹

cd apache-tomcat-8.5.23
1

# 常用命令

# 启动

/usr/local/apache-tomcat-8.5.23/bin/startup.sh
1

# 停止

/usr/local/apache-tomcat-8.5.23/bin/shutdown.sh
1

# 目录内执行脚本

./startup.sh
1
