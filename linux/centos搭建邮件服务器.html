# CentOS 搭建邮件服务器

# 概述

注意：在搭建此服务器之前，请准备一台干净的centos，在安装时，将主机名设置为localhost！！！！！！！！！！不然很有可能失败！

本文中，dns服务器使用的是bind，邮件服务器使用的是sendmail，dovecot。 其中centos的 IP 地址为：192.168.223.131

# 安装配置 DNS

首先，查看系统中是否已经安装了bind,查看命令如下：

rpm -qa | grep bind
1

如果安装了则显示如下：

如果无显示，则表示未安装，输入下面的命令安装：

yum -y install bind
1

安装完成之后就可以开始配置了：

打开/etc/named.conf，找到下图中圈中的地方

vi /etc/named.conf
1

修改为：（注意：只用改这三处，其他别改）

listen-on port 53 { any; };
        listen-on-v6 port 53 { any; };
/*
...
*/
allow-query     { any; };

        allow-update { none; };

# 安装配置 sendmail

首先，查看系统中是否已经安装了sendmail,查看命令如下：

rpm -qa | grep sendmail
1

安装命令：

yum -y install sendmail sendmail-cf
1

进入/etc/mail目录，打开access文件

cd /etc/mail
vi access
1
2

修改为：

Connect:localhost.localdomain           RELAY
Connect:localhost                       RELAY
Connect:127.0.0.1                       RELAY
Connect:zjr.com                         RELAY
Connect:192.168.223.*                   RELAY
1

4
5

在执行如下命令：

makemap hash access.db < access

打开sendmail.mc,找到下图中处： 改为：

DAEMON_OPTIONS(`Port=smtp,Addr=0.0.0.0, Name=MTA')dnl

再执行下面这条命令：

m4  sendmail.mc  > sendmail.cf

# 安装配置 dovecot

首先，查看系统中是否已经安装了dovecot,查看命令如下：

rpm -qa | grep dovecot
1

安装命令：

yum -y install dovecot
1

接着打开/etc/dovecot/dovecot.conf

vi /etc/dovecot/dovecot.conf
1

找到下图中圈出的那条配置，将#删除

进入/etc/dovecot/conf.d文件夹：

cd /etc/dovecot/conf.d
1

打开10-auth.conf,将下图圈起的地方改为：

vi 10-auth.conf
1

disable_plaintext_auth = no
1

再打开10-mail.conf,将下图圈起的地方改为(就是把#去掉）：

vi 10-mail.conf
1

mail_location = mbox:~/mail:INBOX=/var/mail/%u
1

打开10-ssl.conf,将下图圈起的地方改为： 

ssl = no
1

重启sendmail和dovecot服务

systemctl restart sendmail.service


systemctl restart dovecot.service

提示

在设置密码过程中，会提示密码不符合规范等什么鬼的原因，直接忽略，只用两次密码相同即可。

再创建 /home/user1/mail/.imap/INBOX和/home/user2/mail/.imap/INBOX文件夹，如果不能一次创建，请一级一级创建。 在分别进入到两INBOX文件夹，分别执行如下两条命令：

chown -R user1 .
chown -R user2 .
