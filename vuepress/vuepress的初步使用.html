# VuePress的初步使用

# 环境要求

注意

vuepress已经安装成功

# 起步 helloworld


# 新建一个 docs 文件夹
mkdir docs

# 新建一个 markdown 文件
echo '# Hello VuePress!' > docs/README.md

# 启动测试环境
vuepress dev ./docs

# 构建静态文件
vuepress build ./docs

注意

如果你的现有项目依赖了 webpack 3.x，推荐使用 Yarn 而不是 npm 来安装 VuePress。因为在这种情形下，npm 会生成错误的依赖树。

# 安装过程

  • 在F盘下新建test文件夹

  • 在cmd命令台下进入该文件夹，进入之后运行，在本例中以yarn运行

yarn global add vuepress # 或者：npm install -g vuepress
1
  • 另外，安装时间与网速有关耐心等吧！
